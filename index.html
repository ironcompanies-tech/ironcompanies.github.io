Learn Ironscript

Ironscript is a custom, Python-inspired interpreted language for general-purpose systems. Let's dive in!

📌 Basics

# This is a comment

print "Hello World"

🔁 Loops

repeat 5 times:

print "Looping"

🧠 Variables

set name to "Morris"

print name

📦 Lists

set mylist to [1, 2, 3, 4]

print mylist[2]

🔍 If Statements

set x to 5

if x is greater than 3: print "Big"

🔧 Functions

define greet():

print "Welcome!"

greet()

🔂 While Loops

set count to 0

while count is less than 3: print count set count to count + 1
