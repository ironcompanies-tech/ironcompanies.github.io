What is Ironscript?

Ironscript is a futuristic interpreted system programming language designed for full control, blazing speed, and platform-level hacking power. Built by Iron Companies for the future of devs.

Use Cases

  • System-level programming
  • AI architecture (e.g. Diety)
  • Emulators
  • Terminal-based software
  • Low-level automation
